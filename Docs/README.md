# Docs

Welcome to the Banner Studio documentation folder. This space is designed to help developers, product owners, and QA auditors quickly understand the app, its architecture, and the MVP plan.

Use the documents below to get started:

- `Overview.md` — app purpose, main features, and user flows.
- `Architecture.md` — code structure, core modules, and technology stack.
- `MVP-Plan.md` — prioritized roadmap for the SaaS MVP.
- `Auth-Token-vs-Cookie-Analysis.md` — comparison of token vs cookie auth, with recommendation for Banner Studio.
- `Contribution.md` — how to update docs and make changes to the app.
